Обработка "Обновление конфигурации"

Данная обработка предназначена для выполнения процедур первоначального заполнения базы, а так же для выполнения различных операций после обновлений.

Обработка и ее методы вызывается программно при инициализации конфигурации. При выполнении функции ВыполнитьОбновлениеКонфигурации() происходит проверка первого запуска и выполнение первоначально заполнения базы. После обновления вызывается показ формы с указанием изменений новой версии.

При обнаружении обновления при наличии определенных процедур проведения обновления на данную версию - эти процедуры запускаются.

Обработка не имеет интерактивных способов работы, вся работа с обработкой осуществляется только программно.

Обработка не имеет реквизитов, но в различных версиях может содержать в себе макеты для хранения истории изменений в версиях дополнения. Один макет должен нести информацию по одному обновлению. Имена макетов должны называться в виде "ИзмененияВВерсии_1_0_1_2", т.к. вызов макетов происходит автоматически с формированием его имени.
